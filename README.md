Instructions:
=============
On a machine with ruby >= 1.9.2 at command prompt
`gem install rails`
`git clone git@github.com:dbwest/infinitestore.git`
`cd infinitestore`
`bundle install`
`rake db:migrate`
`rails s`

then visit localhost:3000 in your browser, and you should have a working ecommerce site
